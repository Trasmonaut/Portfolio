Projects [2]

A selection of my recent work

TT WEATHER PREDICT

A weather prediction machine learning model that forecasts weather conditions in Trinidad and Tobago using historical weather data. Built with Python and scikit-learn, the model utilizes regression algorithms to predict temperature, humidity, and precipitation levels based on past weather patterns.The web app allows users to predict future weather in Trinidad and Tobago with a 98% accuracy, for different regions of the country.

Java Game Engine

A small, open-source Java 2D game starter you can copy into new projects. It focuses on clarity and minimal, in order to create a strong foundation for building simple 2D games in Java. Features include a game loop, input handling, sprite rendering, and basic collision detection.
